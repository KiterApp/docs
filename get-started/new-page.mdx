## Overview

PromptLoop AI's Batch API v0 allows you to process large datasets, such as CSVs of websites to scrape, using AI tasks. This guide will help you get started with sending inputs, monitoring jobs, and retrieving results.

## Basic Usage

### Sending Inputs

Send a JSONL file containing your input data:

```
POST /launch-job
json input.jsonl
{"data_uuid": "1", "autoloop_input_array": ["https://example.com"]}
{"data_uuid": "2", "autoloop_input_array": ["https://another-example.com"]}
```


### Monitoring Jobs

Check the status of your job:

```
GET /job-status/:id
```

### Retrieving Results

Fetch the results of a completed job:

```
GET /retrieve-job/:id
```

## Key Endpoints

```typescript
@Post('launch-job')
uploadAutoloopData(@UploadedFile() file: Multer.File, @Body() body: {...})
@Get('job-status/:id')
getJobById(): string
@Get('retrieve-job/:id')
retrieveJobById(): string
```
